---

## Technologies Used

**React.js, TypeScript, Redux.js, Firebase, Stripe, Progressive Web Applications (PWAs), Tailwind CSS, Git**

---

## Features

- User authentication with Google Sign-In  
- Real-time inventory management with Firestore  
- Secure and smooth checkout using Stripe  
- Responsive and fast UI using Tailwind CSS and React  
- PWA support for offline and mobile-friendly experience
